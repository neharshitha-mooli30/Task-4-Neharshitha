# Data Visualization Project

## Author
**Neharshitha Mooli**

## Objective
The objective of this project is to analyze e-commerce sales data and create meaningful visualizations to identify trends, customer behavior, and business insights.

## Tools and Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Excel

## Dataset Description
The dataset contains e-commerce order information including products, payment methods, order status, referral sources, and total revenue.

## Project Tasks
- Data Loading and Exploration
- Data Cleaning and Validation
- Revenue Analysis by Product
- Payment Method Analysis
- Order Status Analysis
- Referral Source Analysis
- Data Visualization using Charts

## Visualizations Created
1. Revenue by Product (Bar Chart)
2. Payment Method Usage (Bar Chart)
3. Order Status Distribution (Bar Chart)
4. Referral Source Analysis (Bar Chart)

## Key Insights
- Chair generated the highest revenue among all products.
- Online payment was the most preferred payment method.
- Cancelled orders were higher compared to delivered orders.
- Instagram was the most effective referral source for customer acquisition.

## Conclusion
This project demonstrates how data visualization can be used to transform raw data into meaningful business insights. The analysis helps identify top-performing products, customer preferences, and effective marketing channels, supporting better decision-making.
